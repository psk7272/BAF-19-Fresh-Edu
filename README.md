# 19기신입교육세션과제1_박승혁

## 1. 관심 있는 주제: 저축은행 및 은행 부실 가능성 분석

## 2. 이유 : 금융회사의 안정성과 부실 가능성을 파악할 수 있는 데이터를 선택했다. 저축은행의 경영지표와 실제 파산은행 사례를 함께 살펴보면 금융회사의 건전성을 이해하는 데 도움이 될 것이라고 생각했다.


## 3.1 데이터명 : banklist.csv (kaggle)

이 데이터는 미국 FDIC의 파산은행 목록 데이터이다. 은행명, 도시, 주, 폐쇄일, 인수기관 등의 정보가 포함되어 있어 실제로 어떤 은행이 언제, 어느 지역에서 파산했는지 확인할 수 있다. 이를 통해 은행 파산이 특정 시기나 지역에 집중되는지 분석할 수 있다.

## 3-2. 데이터명 : 예금보험공사_저축은행업권 주요 경영지표 현황_20251231.csv (공공데이터)

특징 : 예금보험공사_저축은행업권 주요 경영지표 현황 데이터  
이 데이터는 국내 저축은행의 주요 경영지표를 담고 있다. 저축은행명, 지역, 총자산, 대출금, 당기순이익, BIS 자기자본비율, 연체율, 고정이하여신비율 등의 항목이 포함되어 있어 저축은행의 규모, 수익성, 재무 건전성을 비교하기에 적합하다.
